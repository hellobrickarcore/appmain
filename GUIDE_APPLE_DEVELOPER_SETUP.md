# Guide: Set Up Apple Developer Account

## Prerequisites

- Apple ID (personal or business)
- Credit card for $99/year fee
- Legal entity information (for business accounts)

## Step-by-Step Instructions

### 1. Visit Apple Developer Program

Go to: https://developer.apple.com/programs/

### 2. Sign In

- Click "Enroll" or "Sign In"
- Use your Apple ID
- If you don't have one, create it first

### 3. Choose Account Type

**Individual:**
- Personal Apple ID
- $99/year
- Your name appears as developer

**Organization:**
- Business/company Apple ID
- $99/year
- Requires D-U-N-S number
- Company name appears as developer

### 4. Complete Enrollment

1. **Personal Information**
   - Full name
   - Email address
   - Phone number
   - Address

2. **Payment Information**
   - Credit card details
   - $99 USD annual fee
   - Auto-renewal (can cancel later)

3. **Agreements**
   - Read and accept Apple Developer Agreement
   - Read and accept Paid Applications Agreement (if selling apps)

### 5. Wait for Approval

- **Individual accounts**: Usually instant (minutes)
- **Organization accounts**: 1-2 business days (requires D-U-N-S verification)
- Check email for confirmation

### 6. Verify Enrollment

1. Go to https://developer.apple.com/account/
2. Sign in with your Apple ID
3. Verify you see:
   - Membership status: Active
   - Membership expiration date
   - Team ID

## What You Get

✅ Access to App Store Connect
✅ Ability to submit apps to App Store
✅ TestFlight beta testing
✅ Developer certificates and provisioning profiles
✅ Access to beta iOS versions
✅ Developer documentation and resources

## Next Steps

Once enrolled:
1. Set up App Store Connect (see GUIDE_APP_STORE_CONNECT.md)
2. Configure code signing in Xcode
3. Start building and testing

## Troubleshooting

**"Enrollment in Progress"**
- Wait 24-48 hours
- Check email for updates
- Contact Apple Developer Support if stuck

**"Payment Failed"**
- Verify credit card information
- Check card has sufficient funds
- Try different payment method

**"D-U-N-S Number Required"**
- Get D-U-N-S number from https://www.dnb.com/
- Free for businesses
- Takes 1-2 business days

## Cost

- **Annual Fee**: $99 USD/year
- **Auto-Renewal**: Enabled by default (can disable)
- **Refund Policy**: No refunds for unused time

## Important Notes

- Membership is annual, renews automatically
- You can cancel auto-renewal anytime
- Membership required to submit apps
- One account can manage multiple apps
- Can add team members (for organization accounts)
