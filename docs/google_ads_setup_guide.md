# Technical Setup Guide: HelloBrick Google Ads

This guide provides the step-by-step instructions for implementing the approved Google Ads strategy for HelloBrick.

## 1. Create a "New App Campaign"
- **Goal:** App promotion
- **Campaign subtype:** App installs
- **Platform:** iOS (since HelloBrick is currently on iOS)
- **Campaign Name:** `HelloBrick_SearchThemes_Global_Apr26`

## 2. Configure Settings
- **Locations:** India, Vietnam, Singapore. (Add Portugal once the App Store availability is confirmed).
- **Languages:** English.
- **Budget:** £5.00–£10.00 per day (to start).
- **Bidding:** Focus on "Install volume." Set a Target Cost Per Install (tCPI) based on your Apple Ads performance (e.g., £0.50 in India, £1.50 in higher-cost regions).

## 3. Implement "Search Themes" (CRITICAL)
Google App campaigns are mostly automated, but "Search Themes" allow you to guide the algorithm.
1.  Navigate to **Ad Groups > Assets**.
2.  Find the **Search Themes** section.
3.  Add the following 10 themes:
    - `lego part identifier`
    - `what to build with my legos`
    - `missing lego piece`
    - `lego building instructions`
    - `lego activities for kids`
    - `brickit alternative`
    - `identify lego bricks`
    - `lego set helper`
    - `bricklink scanner`
    - `how to find lego part number`

## 4. Create a "Custom Segment" (Audience Intent)
This targets users based on their recent Google Search activity.
1.  Go to **Tools and Settings > Audience Manager**.
2.  Click **Custom Segments** > **+ New Custom Segment**.
3.  Name it: `LEGO High Intent & Competitors`.
4.  Select: **"People who searched for any of these terms on Google."**
5.  Add these terms:
    - `Brickit app`
    - `Bricklink`
    - `Rebrickable`
    - `Lego building ideas`
    - `Lego scanner`
6.  Save and apply this segment to your campaign targeting.
